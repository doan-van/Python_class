Pythonの配列（リスト）¶

Pythonでは、配列の代わりに「リスト」というデータ構造を使用します。リストは、複数の要素を順番に格納できる可変長のデータ構造です。

  1. リストの作成 リストは、角括弧 [] を使って作成します。

説明: この例では、3つの果物の名前を含むリストを作成しています。

In [7]:
# リストの作成
fruits = ["りんご", "バナナ", "オレンジ"]
print("果物リスト:", fruits)  # 出力: 果物リスト: ['りんご', 'バナナ', 'オレンジ']

果物リスト: ['りんご', 'バナナ', 'オレンジ']
  1. リストの要素へのアクセス リストの要素には、インデックスを使用してアクセスします。インデックスは0から始まります。

説明: インデックス 0 を使ってリストの最初の要素（りんご）にアクセスしています。

In [8]:
# リストの要素へのアクセス
first_fruit = fruits[0]
print("最初の果物:", first_fruit)  # 出力: 最初の果物: りんご

最初の果物: りんご

数値の配列¶

NumPy（Numerical Python）とは NumPyはPythonの数値計算ライブラリで、高性能な数値計算を効率的に行うための機能を提供します。 特に、大規模な配列や行列の操作が得意で、数学的な計算を高速に行うことができます。

NumPyのインポート NumPyを使用する際は、通常 import 文を使ってインポートします。

In [9]:
import numpy as np
In [10]:
array_1d = np.array([1, 2, 3, 4])
array_2d = np.array([[1, 2], [3, 4]])
print("1次元配列:", array_1d)  # 出力: 1次元配列: [1 2 3 4]
print("2次元配列:\n", array_2d)  # 出力: 2次元配列: [[1 2]

1次元配列: [1 2 3 4]
2次元配列:
 [[1 2]
 [3 4]]

配列の属性 NumPy配列は、サイズや形状を持っています。

In [11]:
print("1次元配列の形状:", array_1d.shape)  # 出力: 1次元配列の形状: (4,)
print("2次元配列の形状:", array_2d.shape)  # 出力: 2次元配列の形状: (2, 2)

1次元配列の形状: (4,)
2次元配列の形状: (2, 2)

Pythonのforループとif文

  1. forループ forループは、リストやタプル、文字列などの要素を1つずつ取り出して繰り返し処理を行うための構文です。

1.1. 基本的な構文

In [14]:
fruits = ["りんご", "バナナ", "オレンジ"]
for fruit in fruits:
    print(fruit)

りんご
バナナ
オレンジ

1.2. インデックスを使用する range()関数を使うと、インデックスを使ってループを回すこともできます。

In [15]:
# インデックスを使ったループ
for i in range(5):
    print("番号:", i)

番号: 0
番号: 1
番号: 2
番号: 3
番号: 4
  1. if文 if文は、条件に基づいて異なる処理を実行するための構文です。

2.1. 基本的な構文

In [16]:
# 条件に基づいて処理を分岐
number = 10
if number > 5:
    print("5より大きい")

5より大きい

2.2. elseとelif elseやelifを使うことで、複数の条件をチェックできます。

In [17]:
# 複数の条件をチェック
number = 5
if number > 5:
    print("5より大きい")
elif number == 5:
    print("5と等しい")
else:
    print("5より小さい")

5と等しい
  1. forループとif文の組み合わせ forループとif文を組み合わせて、特定の条件を満たす要素に対して処理を行うことができます。
In [18]:
# リストの中の特定の条件に合う要素をチェック
numbers = [1, 2, 3, 4, 5]
for number in numbers:
    if number % 2 == 0:
        print(number, "は偶数です")  # 偶数の場合
    else:
        print(number, "は奇数です")  # 奇数の場合

1 は奇数です
2 は偶数です
3 は奇数です
4 は偶数です
5 は奇数です
